# 证件照自动排版工具

本地离线运行的证件照排版程序，支持多种标准证件照尺寸，自动完成缩放、拼版、描边，导出 300 DPI JPG 图片。

---

## 功能特性

| 排版类型 | 画布尺寸 | 照片数量 | 排列方式 |
|----------|----------|----------|----------|
| 一寸排版 | 5寸（1050×1500 px） | 9张 | 3列×3行 |
| 二寸排版 | 5寸（1050×1500 px） | 4张 | 2列×2行 |
| 小二寸排版 | 5寸（1050×1500 px） | 4张 | 2列×2行 |
| 三寸排版 | 5寸（1050×1500 px） | 2张 | 1列×2行 |
| 驾驶证排版 | 5寸（1050×1500 px） | 10张 | 5列×2行 |
| 一寸+二寸排版 | 7寸（2100×1500 px） | 一寸9张+二寸2张 | 左右混排 |

**证件照标准尺寸（300 DPI）：**

| 类型 | 厘米 | 像素 |
|------|------|------|
| 一寸 | 2.5×3.5 cm | 295×413 px |
| 二寸 | 3.5×5.0 cm | 413×591 px |
| 小二寸 | 3.3×4.5 cm | 390×531 px |
| 三寸 | 6.0×9.0 cm | 709×1063 px |
| 驾驶证 | 2.2×3.3 cm | 260×390 px |

## 直接运行（推荐）

### 前提条件

- Python 3.8 或以上版本
- 已安装 pip

### 安装依赖

```bash
pip install -r requirements.txt
```

### 启动程序

```bash
python main.py
```

---

## 打包为可执行文件

### Windows

双击运行 `build.bat`，或在命令行执行：

```cmd
build.bat
```

打包完成后，可执行文件位于 `dist\证件照排版工具.exe`。

### macOS / Linux

```bash
bash build.sh
```

打包完成后，可执行文件位于 `dist/证件照排版工具`。

---

## 使用步骤

1. 启动程序
2. 拖入照片，或点击上传区域选择文件
3. 选择排版类型（单选按钮）
4. 点击「生成排版」
5. 预览确认后，点击「导出图片」保存为 JPG

---

## 技术规格

- 分辨率：300 DPI
- 导出格式：JPG（质量 95）
- 自动方向检测：横版/竖版自动旋转
- 自动描边：深灰色（#505050），2px，方便裁剪
- 完全离线，无需联网
